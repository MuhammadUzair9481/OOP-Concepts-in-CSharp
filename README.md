
# OOP Concepts in C#

## About This Repository
This repository contains implementations of Object-Oriented Programming (OOP) concepts in C#.
Each concept is organized in a separate folder with example problems and explanations.

## Concepts Covered
- Data Types
- Variables
- Loops
- ArraysArray(1D, 2D , 3D)
- Classes
- Inheritance
- Polymorphism
- Abstraction(abstract classes, interface)
- Encapsulation

## Folder Structure
Each concept folder contains:
- 3 problem-based C# programs
- 1 README.md explaining the concept

## Purpose
This project is created for learning and practicing OOP concepts and for academic submission.
